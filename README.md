### plAnt – an application for developers and QA engineers to facilitate bug discovery

> The presented system enables developers and testers to collaborate through a specialized platform functioning as a marketplace for mobile application testing.
The system facilitates faster and more efficient detection, documentation, and resolution of defects, significantly reducing the time required to deliver high-quality mobile applications to the market.

This application was developed as part of the Bachelor’s thesis in *Information Systems and Technologies (09.03.02)* at [RTU MIREA](https://english.mirea.ru/), by Alexandr Chekunkov.
The thesis was successfully defended with the grade **“Excellent”**.
